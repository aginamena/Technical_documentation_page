The Software Behind Motion Sensors (If Statements)

A computer program must make choices on which way to proceed when processing information. The outcome of the choice is based on the condition(s) given. Conditions are expressions that are true or false[1]. For example,

if the light is turned on, turn it off. If the light is turned off, turn it on. If all the data has been received, process the data, or else (if all the data has not been received) wait for all data to come in.

In computer programming, making decisions when given specific conditions is called if statements[2]. If statements are used to compare two or more options. For example,

if a student obtains good grades, he/she is likely to go to college.

The only condition required to go to college is to achieve good grades. If a student does not obtain good grades, then he/she will not be accepted into college. These two conditions, good grades and poor grades, are specific conditions that are either true or false. If the condition is false (poor grades), the student will not get into college. If the condition is true (good grades), the student will get into college.

Conditions also known as Boolean expressions have a value of true or false[4]. If the value is false, the else code will be executed, as shown in the in the Figure above. If the Boolean expression is true, the if statement will be executed.

Another use of an if statement can be to test if a password is correct or not. When someone enters a password to unlock a phone, the registered password in the phone is compared with what was entered to see if they are the same. If they are the same, then the phone unlocks. If they are not the same, the system will prompt the user to enter the correct password. Here is a written pseudocode of an if statement to determine if a password is correct or not.

if(password = hello)
  display 'password accepted';
else
  display 'Sorry, the password is incorrect';

Looking at the pseudocode above, the password the user enters is compared with “hello” to see if they are the same. Depending on the Boolean condition, one of the statements will be displayed on the screen of the phone (“Password accepted” or “Sorry, the password is incorrect”).

Uses of if statements

If statements are used to make decisions given a specific condition. If statements operate by using Boolean expressions (true or false value). Given two or more options, if statements pick one of the options. It (if statments) could either display password accepted or sorry, the password is incorrect based on our input as shown in previous example.

Motion Sensors at Home

Motion sensors save energy costs by turning the light on and off when motion is detected. This is due to if statements in the sensor’s software.
The example below shows how motion sensors work using if statments

if(motion is detected)
 Turn on light;
else
  Turn off light;

The if statement in the example above checks if motion is detected. If motion is detected, the light bulbs will turn on. If motion is not detected, the lights will remain off. Since the motion sensors will be placed at home, especially in places with less movement, like the parking garage, lundry room and so on, the lights should come on only when it detects motion, by day or by night. Energy should only be used when the sensors detect motion. Thus, saving energy costs at home

Ways in which Motion Sensors Save Energy Costs

With the understanding of how if statements work, if statements can be used in motion sensors to reduce energy costs at home by turning the light bulbs on when the sensors detect movement.

An experiment was carried out in Indonesia at Universitas Budi Luhur to show that installation of motion sensors can contribute to a reduction of electric consumption[6]. These sensors were installed in washrooms to prove that electricity consumption from the lights can contribute to the reduction of energy costs.
